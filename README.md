# my-custom-skills
A personal collection of custom Claude skills, built for specific business and productivity use cases.

## 📦 Skills List
| Skill Name | Description | File |
|------------|--------------|------|
| Asset Investment Report | A custom Claude skill for generating professional asset investment analysis reports, including market trend analysis, portfolio evaluation, and investment recommendations. | `asset-investment-report.skill` |

## 🚀 Usage
1.  Download the `.skill` file from this repository
2.  Import it directly into Claude Desktop / Claude.ai
3.  Activate the skill and start generating professional investment reports

## 🧩 How to Add New Skills
1.  Create your custom Claude skill file
2.  Upload to this repository via `Add file > Upload files`
3.  Update this README with the new skill's information
